
This folder runs the code for stylistic cleanup as an AI-detector avoidance technique. It applies the stylistic cleanup to the dataset in `HC3-Dataset-Samples/hc3_unified_1000_seed42.csv`

1) Run the `final_project_stylistic_cleanup.py` file to generate the perturbed csv 
2) Or, directly use the `hc3_sample_stylistic_cleanup.csv`
3) Feed the perturbed CSV into the AI detection models to generate new test results
